| **operator** | **use**                 |
| ------------ | ----------------------- |
| `=`          | assign a value          |
| `+=`         | add and assign          |
| `-=`         | subtract and assign     |
| `*=`         | multiply and assign     |
| `/=`         | divide and assign       |
| `//=`        | floor divide and assign |
| `%=`         | modulus and assign      |
| `**=`        | exponentiate and assign |
